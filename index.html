СХЕМОТЕХНІКА ТА ЕЛЕКТРОНІКА

Фундамент для будь-якого Hardware-інженера

⚡ ЩО МИ ВИВЧАЄМО

Схемотехніка — це мистецтво поєднання електронних компонентів для створення функціональних пристроїв.

  • Закон Ома та правила Кірхгофа — база для розрахунку кіл.
  • Напівпровідники: діоди, транзистори (MOSFET, BJT).
  • Цифрова логіка: AND, OR, NOT, тригери, лічильники.
  • Проєктування друкованих плат (PCB Design).

ЗАПИСАТИСЯ

ВБУДОВАНІ СИСТЕМИ (EMBEDDED)

Програмування, що вдихає життя в залізо

💾 МІКРОКОНТРОЛЕРИ

Ми не просто пишемо код, ми керуємо напругою на ніжках процесора. Це "bare metal" програмування.

Основні платформи:

// Приклад коду C++ для STM32 HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, GPIO_PIN_SET); HAL_Delay(1000); HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, GPIO_PIN_RESET);

Ви навчитесь працювати з регістрами, перериваннями та інтерфейсами передачі даних (I2C, SPI, UART).

🎬 ВІДЕО: C++ ДЛЯ EMBEDDED

📺 Переглянути на YouTube

C та C++ — це основні мови для розробки під мікроконтролери через їх швидкодію та контроль пам'яті.

ПРИЄДНАТИСЯ

INTERNET OF THINGS (IoT)

Мережа фізичних об'єктів, оснащених сенсорами

📡 ЕКОСИСТЕМА IoT

IoT поєднує embedded-розробку з передачею даних. Ваші пристрої стають частиною глобальної мережі.

  • Сенсори: Температура, вологість, GPS, акселерометри.
  • Зв'язок: Wi-Fi (ESP32), Bluetooth BLE, ZigBee, LoRa.
  • Протоколи: MQTT, CoAP.

🔍 СКЛАДОВІ РОБОТА

Робототехніка — це синергія механіки, електроніки та програмування.

  • Механіка: Корпуси, приводи, маніпулятори (3D-моделювання).
  • Сприйняття: Комп'ютерний зір (OpenCV), лідари.
  • Керування: PID-регулятори, нейромережі.

СТВОРИ СВОГО ПЕРШОГО РОБОТА

РОЗПОЧАТИ

АРХІТЕКТУРА КОМП'ЮТЕРНИХ СИСТЕМ

Як працює процесор зсередини?

Вивчення того, як транзистори перетворюються на логіку, а логіка — на обчислення.

Ключові теми:

  • Архітектури: x86, ARM, RISC-V.
  • Пам'ять: Регістри, Кеш (L1/L2), RAM.
  • FPGA (ПЛІС): Програмовані логічні матриці (Verilog/VHDL).
